# Complejo Nuclear Avanzado - Sitio Web

Sitio web moderno y futurista para presentar un complejo nuclear subterráneo a inversionistas.

## Características

- Diseño futurista con Anduril-style
- Animaciones suaves y dinámicas
- Secciones interactivas
- Galería de renders
- Formulario de contacto
- Especificaciones técnicas
- Totalmente responsive

## Estructura

- `index.html` - Estructura HTML
- `style.css` - Estilos y diseño
- `script.js` - Interactividad

## Colores

- Verde Nuclear: #00ff41
- Cyan: #00d9ff
- Fondo Oscuro: #0a0e27
- Texto Claro: #e0e0e0

## Cómo Usar

1. Abre `index.html` en tu navegador
2. Personaliza el contenido con tus datos
3. Reemplaza los placeholders con tus imágenes
4. Actualiza la información de contacto

## Personalización

Todos los datos se pueden editar en `index.html`:
- Nombre del proyecto
- Especificaciones técnicas
- Información de contacto
- Imágenes y renders
